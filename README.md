## 📦 Installation

### Prerequisites

Make sure you have Node.js (v18 or higher) and npm installed on your system.

```bash
node --version  # Should be v18 or higher
npm --version   # Should be v9 or higher
```

### Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/bilalshabbir31/Portfolio.git
   cd Portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   ```
   Navigate to http://localhost:5173
   ```

## 🎨 Color Scheme

The portfolio uses a modern dark theme with the following color palette:

- **Text Colors**: 
  - Primary: `slate-100`

## 🌐 Deployment (GitHub Pages)

This site deploys automatically to GitHub Pages when you push to `main`.

1. On GitHub, open **Settings → Pages**
2. Under **Build and deployment**, set **Source** to **GitHub Actions**
3. Push to `main` (or run the **Deploy to GitHub Pages** workflow manually)
